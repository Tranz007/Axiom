# Code Of Conduct

Axiom is an experimental open-source project for people working on agent-facing software, policy, safety, and developer tooling.

## Expected Behavior

- Be respectful and direct.
- Critique ideas and code, not people.
- Assume good faith unless behavior proves otherwise.
- Keep discussions focused on making Axiom clearer, safer, and more useful.
- Welcome contributors who are learning the codebase or the problem space.

## Unacceptable Behavior

- Harassment, threats, or personal attacks
- Discriminatory language or conduct
- Publishing private information without consent
- Repeatedly derailing technical discussions
- Using project spaces for spam or unrelated promotion

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; block users; or take other reasonable action to keep the project constructive.

Security issues should follow [SECURITY.md](SECURITY.md), not public issue threads.
